# Block Breaker 🧱

A mobile-friendly Block Breaker game built with HTML5 Canvas — no build tools required.

## Play Online

👉 **[Play now on GitHub Pages](https://ishana23g.github.io/block_breaker/)**

## How to Play

| Control | Action |
|---------|--------|
| **Swipe / drag** (mobile) | Move the paddle |
| **Tap** (mobile) | Launch the ball |
| **Move mouse** (desktop) | Move the paddle |
| **Click** (desktop) | Launch the ball |

- Break all the blocks to advance to the next level.
- You have **3 lives** — don't let the ball fall off the bottom!
- Some blocks (on higher levels) take **2 hits** to destroy.
- Score bonus is awarded for completing each level.
- There are **5 levels** in total, each faster and with more blocks.

## Deploy on GitHub Pages

1. Go to **Settings → Pages** in this repository.
2. Under *Source*, select **Deploy from a branch**.
3. Choose **main** (or your branch) and **/ (root)**.
4. Click **Save** — your game will be live at `https://<username>.github.io/block_breaker/`.

## Run Locally

Just open `index.html` in any modern browser — no server needed.

```bash
open index.html   # macOS
xdg-open index.html  # Linux
```
